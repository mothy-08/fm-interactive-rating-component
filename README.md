## What I learned

- Used `history.pushState()` and the `popstate` event to handle back-button navigation within a single-page component—so pressing back resets to the rating form instead of leaving the page. 
- Used HTML's native `hidden` attribute to toggle visibility instead of relying on custom data attributes or CSS classes.
